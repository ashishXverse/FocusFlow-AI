# 🚀 FocusFlow AI

### Smart AI Productivity Assistant powered by Gemini

FocusFlow AI is an intelligent productivity dashboard that helps users organize tasks, prioritize work, visualize productivity, and receive AI-powered recommendations using Google's Gemini API.

🌐 **Live Demo:**  https://focus-flow-ai-project.web.app/

---

## 📌 Problem Statement

Managing multiple tasks manually often leads to missed deadlines, poor prioritization, and reduced productivity.

FocusFlow AI solves this by combining task management, analytics, and Artificial Intelligence to help users focus on what matters most.

---

## ✨ Features

- ✅ Create, delete and manage tasks
- ✅ Mark tasks as completed
- ✅ Progress tracking
- ✅ Dashboard analytics
- ✅ AI-powered productivity recommendations using Gemini
- ✅ Local Storage persistence
- ✅ Responsive modern UI
- ✅ Firebase Hosting deployment

---

## 🤖 AI Productivity Coach

Using Google's Gemini API, the application can:

- Analyze pending tasks
- Identify the highest-priority task
- Recommend an execution order
- Estimate workload
- Provide productivity tips
- Suggest an optimized work schedule

---

## 📊 Dashboard

The dashboard includes:

- 📋 Total Tasks
- 🔥 High Priority Tasks
- ✅ Completed Tasks
- ⏳ Pending Tasks
- 📈 Progress Bar
- 📊 Analytics Charts
